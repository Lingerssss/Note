find grep more cat tail head

1.

find

. /

-name -size +10M -type d(目录) f(file) -ctime(时间天) -cmin(时间_分)

-amin(读过的文件)

-maxdepth 3 指定查找文件深度为3

-print 0 | xargs -0将输出打印为1行


2.

grep

grep -n "aa" *.txt 在所有txt中查找aa -n是显示行号 -r递归子路径

-A(after) 2 输出目标及后面两行

-B(before) 2输出目标及前面两行

3.

cat


4.

more 只查看一部分 -3(每一页行数) +4(从第4行开始读)

空格翻页 B往前翻页



5.

head tail

head查看前10行 -n指定查看多少行

tail -f 查看最后10行，-f持续监听文件变动
